# sk5short 是什麼
sk5short用來製作短連結，用javascript來重新導向，可用github page、gitlab page等架設，架設完成可以有類似bitly、youtube短網址的功能。

看看[快速開始](quickstart.md)來了解更多關於使用的方法吧！

# 特點
  * 可以在各種常見免費網站架設服務上使用。
  * 可以更改跳轉網址或更換成其他網頁。
  * 方便操作，有圖形介面，清楚明瞭。
  * 可以自訂連結預覽照片、標題、敘述。（facebook）
